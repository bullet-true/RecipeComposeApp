# Recipe Compose App - приложение для рецептов с использованием Jetpack Compose

## Технологический стек:

### **UI/UX разработка:**

*   **Jetpack Compose**
*   **Material Design 3**
*   **State Management**
*   **Coil**

### **Архитектура приложения:**

*   **MVVM паттерн**—
*   **StateFlow**
*   **Repository Pattern**
*   **Модульная структура**

### **Работа с данными:**

*   **Retrofit + OkHttp**
*   **Room Database**
*   **Kotlin Coroutines**
*   **JSON parsing**

### **Production-инструменты:**

*   **Hilt (Dagger)**
*   **Git + GitHub**
*   **Pull Request**
*   **JUnit + Mockk**
